Creates a new sample Vaadin application as a multi-module Maven project. Illustrates how to use Vaadin to create CRUD application.

This project is not regular template project but rather a sample CRUD example project and it's also available under Vaadin Samples (Samples/Vaadin) category.

Several Maven modules will be generated and two of them (Maven Web projects) can be used to run application. One is for production mode and the second one is for for development. See the readme file in the generated project for more details.
